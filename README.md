# O Mago da Adivinhação

Um projeto teste criado com fins educativos e para treino de linguagem. Consiste em um jogo de adivinhação interativo, utilizando lógica matemática como base.

## Objetivos
* Prática da estrutura e funcionamento do JavaScript.

## Tecnologias
* Linguagem: HTML, CSS e JavaScript

## Instalação e Uso
1. Clone o repositório ou baixe os arquivos.
2. O projeto não exige instalações externas, basta apenas executar o arquivo "index.html".
